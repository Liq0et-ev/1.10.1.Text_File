# 1.10.1.Text_File
# Stipendijas saņēmēju atlase no teksta faila 

Programma nolasa teksta faila saturu, kur katrā ierakstā ir norādīts studenta vārds, uzvārds un atzīmes. <br>

Tālāk programma atlasa tikai tos studentus, kuriem pienākas stipendija: nav nevienas atzīmes zem 7 un septītnieku skaits nav lielāks par 2. <br>

Ekrānā katrs atbilstošais students tiek izvadīts atsevišķā rindā formātā: `Vārds Uzvārds 7nieku_skaits`. <br>

Vārds, uzvārds un septītnieku skaits tiek atdalīti ar vienu atstarpes simbolu. <br>

Ievades kļūdas ir atļauts neapstrādāt. <br>
